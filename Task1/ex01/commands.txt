Название: alias
Описание: Создание псевдонимов для команд в терминале
Синтаксис: alias имя='команда'
Пример: alias ll='ls -la'
Примечание: Чтобы упростить выполнение часто используемых команд

===
Название: grep -r
Описание: Поиск текста по шаблону в файлах или выводе других команд
Синтаксис: grep [опции] шаблон [файлы]
Пример: grep -r "ERROR" ./logs
Примечание: Можно искать с помощью регулярных выражений. -r - рекурсивный поиск

===
Название: sort
Описание: Сортировка строк в текстовом потоке или файле
Синтаксис: sort [опции] [файл]
Пример: sort -n numbers.txt
Примечание: Ключ -n сортирует по числам, -r — в обратном порядке

===
Название: tee
Описание: Сохраняет вывод команды в файл и одновременно выводит его на экран
Синтаксис: команда | tee [опции] файл
Пример: ls -l | tee files.txt
Примечание: Опция -a добавляет в файл вместо перезаписи

===
Название: time
Описание: Измерение времени выполнения команды
Синтаксис: time команда
Пример: time ls -R /
Примечание: Показывает реальное, пользовательское и системное время

===
Название: watch
Описание: Периодический запуск команды и отображение её вывода
Синтаксис: watch [опции] команда
Пример: watch -n 2 df -h
Примечание: Опция -n задаёт интервал обновления в секундах

===
Название: jobs
Описание: Просмотр списка фоновых и приостановленных процессов текущей оболочки
Синтаксис: jobs [опции]
Пример: jobs -l
Примечание: -l показыет PID для каждого процесса. Используется с fg, bg чтобы потом перевести процессы на передний план или в фоновый режим.

===
Название: crontab
Описание: Планировщик задач для периодического запуска команд в фоновом режиме
Синтаксис: crontab [опции]
Пример: crontab -e
Примечание: -e чтобы редактировать запись. Записи в crontab выглядят так: minute(s) hour(s) day(s) month(s) weekday(s) command(s)

===
Название: iftop
Описание: Мониторинг сетевой активности в реальном времени
Синтаксис: iftop [опции]
Пример: sudo iftop -i eth0
Примечание: Аналог top для сетевого трафика, требует root-прав

===
Название: iostat
Описание: Статистика использования процессора и ввода-вывода устройств
Синтаксис: iostat [опции] [интервал [количество]]
Пример: iostat -dx 2 5
Примечание: Помогает выявлять узкие места в дисковой подсистеме
